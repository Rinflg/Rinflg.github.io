电子版书本

​ 数学中有很多例子是无法单纯依靠最初的直觉来理解的。本章我们会从罗素悖论开始，说明依靠经验或者直觉判断而导致假命题被当做真命题是一件多么容易的事情。之后我们会讨论集合论和拓扑学中的一些必要结果，然后进入本章的核心-概率的基础。

2.1 罗素悖论

​ 有时，数学家苛求的严谨会让人感到奇怪。因为一些结论看似“显然成立”，但还是会有数学家没完没了地证明它们。

​ 罗素悖论与集合有关。我们都知道，如果P是某种性质，那么全体具有性质P的对象就自然构成一个集合。

​ 罗素提出了这样一个集合： R = { x|x∉x }

那么R是集合吗？如果是，它的元素是什么？对表达式x∉x，我们以R为例容易发现奇怪的结果。如果R∉R，那么R就符合了自身集合的条件，它将属于R，即R∈R。但如果R∈R，那么R就不符合自身集合的条件，它将不属于R，即R∉R。两个情况结合起来，我们会发现，当R∉R时，一定有R∈R。

这显然是荒谬的，我们可以得出结论：通过简单地收集具有给定性质的所有对象来形成集合是不可行的。松散定义和非正式陈述是危险的。

2.2 集合论综述

​ 在描述概率的一般规律之前，我们快速地了解集合论和拓扑学中的一些知识。它们对学习概率语言至关重要，而概率语言是理解整门学科的重要部分。这部分内容中的一些可能在离散里已经涉及过，但稳中起见还是再复习一下。

​ 如果想要讨论事件的概率，那么自然会涉及取定某些大型集 Ω并对其子集指定概率的问题。但令人惊讶的是，不存在这样通用的方法，可以将概率分配给每个子集，并使得概率函数满足某些“自然”条件。这与巴拿赫-塔尔斯基悖论有关，我们会在2.6节简单地讨论。以防不能将概率一致地分配给所有可能的子集，我们必须留意要考察的事件。为此，我们需要集合论和（点集）拓扑中的一些基本事实。这些基本关系就足以应对概率论入门课。

​ 我们从一些基本的定义和集合性质开始。

​ 令A，B，C，….表示集合，并让a，b，c，…分别表示这些集合中的元素。如果a是A中的一个对象，那么我们就说a∈A；如果a不包含在A中，那么a∉A。但通过验证性质我们往往很难确定某个元素是否属于一个集合。例如：如果A是所有至少能写成两个素数之和的偶数的集合，那么有一个简单的方法来验证某个给的的数是否属于A，但计算成本很高。容易看出4=2+2,100=47+53且1864=3+1861； 那么24601^(20131701!)+2013!^24601*1701!+3^(2012!)呢，可能验证起来就会有些难度。哥德巴赫猜想可能能帮助我们解决这个问题，但是其证明还需要很大的努力。

如果说A的每个元素都在B中，我们就说A是B的一个子集，并记作A⊆B，也可以说B是A的超集。

最重要的集合之一是空集，它是唯一没有元素的集合。我们把空集记作∅。

空集显然是独一无二的，但它有很多不同的表示方法。由此，我们希望知道如何证明两个不同描述下的集合相等。一种常用的方法是：要想证明A=B，只需要证明A⊆B且B⊇A。

并集：由A和B中所有元素构成的集合。记作A∪B。

交集：由同时存在于A和B中的元素的集合。记作A∩B。

补集：由全体不存在于A中的元素组成。记作A^c。

笛卡尔乘积：它是全体有序对（a，b）的集合，其中a∈A并且b∈B。特别地，当A=B时，A^2表示A×A。当有n个A相乘时，记作A^n。需要注意的是这里的元素都为有序对，（1,3）和（3,1）是完全不同的两个元素。

幂集：记作p（A），是A所有子集的集合。如果A={x，y}，那么p（A）={Ø，{x}，{y}，A}

2.2.1 无穷大的大小和概率

集合论的一个重要结论是存在不同程度的无穷大。这一节我们着眼于讨论无穷大的阶以及概率的含义。这些细节能帮助我们理解为什么某些集合是值得研究的。

我们首先还是复习一些概念。

单射函数：如果函数f：A->B将不同的输入对应到不同的输出，那么称f是一对一的函数，又叫单射函数。这样以来，如果f是一个单射函数，那么f（x）=f（y）是x=y的充要条件。

满射函数：如果对于任意给定的b∈B，都存在一个a∈A能够使得f（a）=b，那么就称f：A->B是满射的。也称为映上的。

双射函数：既是单射又是满射的函数称为双射函数。

集合的大小（基数）：集合的大小依次为有限、可数和不可数。如果集合A中的元素与集合{1,2,3….n}之间存在双射关系，那么就说A是大小为n（或基数为n）的有限集。如果说在A和自然数集存在一个双射函数，那么A是可数的，我们将它的大小记为阿列夫零ℵ₀。如果A既不是有限的，也不是可数的，那么我们就说它是不可数的。集合A的大小记作|A|或者#A。可数集的大小为无穷大，并且是最小的无穷大。

比较反直觉的一点是，即便A是B的一个真子集，即A⫋B，|A|=|B|的可能也是存在的。举个最简单的例子即正偶数集E与正整数集P。显然E⫋P，但我们可以选择双射f（x）=x/2，f：A->B来满足等势的充要条件。也就是E和P有相同大小。

事实上，正整数集，整数集Z，有理数集Q乃至全体n元有理数组的集合Q^n都具有相同的大小，它们都是可数的。而实数集R，平面R^2以及n维空间R^n都是不可数的。关于这里的证明，我们使用的是康托尔的对角化论证。

2.2.2 开集和闭集

我们在微积分里谈及过它们的概念。

直线上常见的区间共有四种：

    1. [a,b]:={x∈R|a≤x≤b}。它包含了两个端点，被称为开区间。
    2. [a,b):={x∈R|a≤x＜b}
    3. (a,b]:={x∈R|a＜x≤b}
    4. (a,b):={x∈R|a＜x＜b}。它不包含两个端点，被称作开区间。

第二、三种被称为半开/闭区间。

当我们需要升维时，选择笛卡尔乘积是个好想法。通过集合[a,b]×[c,d]:={(x,y)|a≤x≤b，c≤y≤d}可以考察二维空间中的矩形。类似地，集合[a,b]×[c,d]×[e,f]:={(x,y,z)|a≤x≤b，c≤y≤d,e≤z≤f}可以用来考察三维空间中的立方体。为了保持符号的整洁，我们可以用下面的形式来描述R^n

[a1,b1]×…..×[an,bn]={(x1，….，xn)|ai≤xi≤bi}

当然，矩形或者立方体并不是唯一的选择。平面中的圆，三维空间中的球都是很有用的。比起矩形能够更好地组合在一起，圆和球则更加便于理论计算。

实践中，我们研究概率时的对象集合维度可能很高。比如一个经济模型，它可以有10个参数，而我们想研究导致某个特定结果出现的参数值的大小或概率。此时，我们就会想要寻找一些在所有维度上都通用的符号来使得研究过程更加方便。这些符号也能为我们学习概率提供一个真正坚实的基础。

开球：半径为r，球心为a=（a1，….，an）∈R^n的开球定义为 Ba（r）:={x=(x1,….,xn)|(x1-a1)^2+…..+(xn-an)^2<r^2}

闭球：定义方法类似，将＜换成≤即可。

空间中的开、闭集：设A⊆R^n，如果对于任意给定的a∈A，都存在一个r使得Ba（r）⊆A，那么就说集合A是开集。如果A的补集是开集，那么A就是闭集。

至此，关于术语的部分就暂时结束了。学习定义大概率不是一件愉快的事情，但它的价值在于使我们能够与他人清晰、有效地沟通。虽然不一定是立刻，但我们的学习过程中总会看到它的价值。

2.3 结果空间、事件和概率公理

​ 本节的内容开始时将会比较简单，但其内容会随着我们所考察的问题越来越微妙而在技术层面上越来越困难。一些问题或许会要求我们用到分析学和点集拓扑的知识。虽然我们之前讨论过罗素悖论，但不可否认很多时候日常经验虽然有误导我们的风险，却也可以（甚至是多数情况下可以）为我们提供帮助。而出现风险的地方常常是无穷大——要么是无穷多个集合，要么是有无穷多个元素的集合。总之，只有出现无穷大，我们就应当非常小心。但反之，如果说是有限多个集合，这些集合中也包含有限多个元素，那么我们通常可以大胆地相信自己的直觉。具体的技术细节还会在2.6节详细讨论。

​ 我们先来试图将一些常用的直观概念形式化。为了合理地谈论概率，我们需要一些信息。首先必须指出所有可能的结果以及每个结果发生的概率。

​ 我们假设所有可能的结果都是某个给定集合Ω的子集。因此，Ω可能就是正整数集，事件就是抛出“正面”之前，抛硬币的总次数。另一个例子是，假设Ω是一个单位圆，事件是单位圆的子集。此时，如果我们向单位元中投掷飞镖，那么它就要在某处着陆，而圆周和圆内的每个点都是可能的着陆点。

​ 这样，我们可以把Ω称为样本空间或结果空间，并把Ω中的元素称作事件。这个定义在很多情况下都适用。当Ω是有限集或者可数集时，这个定义可行。但对于一般的概率空间，这个定义就需要进行一些修改。一旦有了结果空间Ω，我们就会想为Ω中的不同元素指定概率。为此，我们引入概率函数，并把它记作Prob。我们用Prob（A）来表示事件A发生的概率，使用时通常把它简写成Pr（A）。

​ 举例来说，假设一个老虎机的第一个轮盘上有20个符号：10个梅花、5个红桃、3个方块和2个黑桃。当我们玩的时候，如果这20个符号中的每个都等可能地出现，那么样本空间和概率函数分别是什么？

​ 这个问题并不复杂，但仍有一些微妙之处值得我们注意。由于这里的10个梅花看起来都一样，因此我们可以不去详细区分他们。当轮盘停止时，如果我们看到一个梅花，那么我们无法也没有必要去区分它第1个还是第10个。

​ 样本空间是所有可能结果都集合，所以我们有 Ω={♠，♥，◇，♣}。

​ 20个对象中的每个都等可能地出现，因此每种符号出现的概率就是具有这种符号的对象的个数除以对象的总数。即：Pr（♠）= 2/20 ， Pr（♥）= 5/20 ，Pr（◇）= 3/20，Pr（♣）=10/20。

​ 我们会发现有两条简单的性质：（1）每个概率都是非负的，且至多为1 （2）不相交的概率函数之间具有关系时，可以相互推理：Pr（♣或♥） = 15/20 = Pr（♣）+ Pr（♥）

​ 实际上，最后两个性质是我们希望概率函数能够服从的一般特性。具体来讲，我们希望概率函数能够满足下面的愿望列表

愿望列表

（1）对于任意的事件A，均有 0 ≤ Pr（A）≤ 1。如果 Ω 是结果空间，那么Pr（Ω）= 1

（2）如果 {Ai} 是一族两两互不相交的集合（也就是说，当 j ≠ k时 Aj ∩ Ak 是空集），那么 Pr（UiAi）=∑iPr（Ai）

​ 这些条件看起来是显然的，但事实上，在问题涉及无穷大时，这个条件并不能满足。因此，为了保留这个属性，在分配事件的概率时要更加小心，具体我们会在下一节中讨论。

​ 由这两个条件，我们可以推出一个结果。

​ Pr（Ø）= 0

​ Proof：Ø ∩ Ω = Ø -> Pr( Ø ∪ Ω ) = Pr（Ø）+Pr（Ω），即 1 = 1 + Pr（Ø） -> Pr（Ø）=0

​ 如果事件A和B相互独立，即A和B不以任何方式互相影响时，A与B同时发生的概率Pr（A∩B）=Pr（A）*Pr（B）。

2.4 概率公理

​ 现在，我们尝试给出概率函数满足的基本规则。我们暂时只考察一些重要的特殊情况，因为更一般的情况可能需要用到一些高等分析学的知识。在我们规定的有限或可数的结果空间中，结论都相当简单。而当无穷大出现时，直觉和经验就可能对我们产生误导。

​ 关于更一般的部分，我们可以先给出柯尔莫戈洛夫的概率公理。现在可以了解其存在和合理性，而关于其更详细、严格的讨论和σ代数Σ中到底包含了什么，我们可以在2.6中再去了解。

​ 柯尔莫戈洛夫的概率公理：Ω是一个结果空间，Σ是一个σ代数。如果概率函数满足下列条件，那么（Ω、Σ，Prob）就是一个概率空间.

​ 1.如果A∈Σ，那么Pr（A）是有定义的，并且0 ≤ Pr（A）≤ 1

​ 2.Pr（Ø）= 0 且 Pr（Ω）=1

​ 3.设 {Ai}是由有限个或可数个两两互不相交的集合构成的集族，并且每一个集合都是Σ中的元素，那么Pr（UiAi）=ΣiPr（Ai）

​ 其中，第一条说明了我们可以对σ代数Σ中的任何事件分配概率，并且概率必须在0和1之间取值。

​ 第二条公理断言，任何事情都不发生的概率为0，有事情发生的概率为1

​ 最后一条公理中出现了关键词“有限”和“可数”。它第一次提及了无穷大。事实上，在后续的学习中我们会发现，把结论限定在有限集或可数集上是很重要的，在更大的无穷大中讨论会带来严重的问题。

​ 我们来探讨一些概率空间的例子，然后再下节中继续讨论这些空间的重要性质。

​ 概率空间中最好的，也是最重要的例子之一是：Ω是一个有限集，记作Ω={w1，w2，…..，wn}；σ代数是这个有限集Ω的所有子集的集合。我们需要对Ω中的元素分配概率，最简单的方法就是让所有元素的概率都相等，即对于任意的 1 ≤ k ≤ n，有 Pr（Wk）=1/n。此时，如果A是Ω的任意一个子集，那么Pr（A）就是 |A|/|Ω|。这通常被称为计数模型，在这种概率定义下，所有公理都成立。

比如，抛掷一颗质地均匀的骰子。此时，我们可能投掷出的结果有1、2、3、4、5、6，其中每种结果出现的概率都是1/6。设A={1、3、5}，那么Pr（A）=3/6=1/2，即我们有50%的概率投掷出奇数。

关于概率相加的问题，我们也要注意考察交集是否为空。比如A={2、4、6}（抛掷出偶数）B={3、5}（抛掷出奇素数）C={2、3、5}（抛掷出素数）。此时，Pr（AUB）=Pr（AUC）=Pr（A）+Pr（B）≠Pr（A）+Pr（C）。

在求解时，也有一些比较简单易行又容易理解的思路。以下面的题目为例子：

此时有一枚硬币，我们连续抛掷5次，此时的结果空间中共有2^5=32种可能。用A表示“出现偶数次正面”这一结果。求Pr（A）

​ （1）比较老实粗暴的方法：罗列所有的可能结果

​ 利用计数法，我们看到在事件列表中，32个结果中共有16个结果出现了偶数次正面。即Pr（A）=16/32=1/2

​ （2）一个错误的方法

​ 出现偶数次正面的情况有三种，即0、2或4次正面；出现奇数次正面的可能性也有三种，即1、3或5次正面。因此，Pr（A）=1/2

​ （2）对称法

​ 事实上，此处的反面与正面是完全等价的。如果我们记此时的正面为反面，原来的反面也就自动变成了正面。也就是说投掷出奇数次正面的可能性应该等于投掷出奇数次反面的可能性。而总次数为5次，二者必有一事件发生概率和为1。所以Pr（A）=1/2。

2.5 基本概率规则

​ 关于概率空间，也有一些基本规则。这些结论可以由前面的三条公理推知。需要强调的是，我们不是把概率分配给所有事件，而是只对特殊集合分配概率。更严格的论述留到2.6节，我们先来简单陈述一些比较直观的事实。

​ 给出一个概率空间（Ω，Σ，Prob），其中Σ是由Ω的全体子集构成的集合。Prob把概率分配给Σ中的每一个元素。特别有，Pr（Ø）=0，Pr（Ω）=1，并且当A⊆Ω时，Pr（A）是有定义的。并且，Σ中有限个或者可数个两两互不相交的元素的并的概率就等于这些元素的概率之和。下面是四条非常好用的性质，值得我们单独记忆。

设（Ω，Σ，Prob）是一个概率空间，可以得到的结论有：

​ （1）“全概率公式”：如果A∈Σ，那么Pr（A）+Pr（A^c）=1

​ 注：这里的全概率公式是更一般的全概率公式的一个特例，因此带引号（“”）。

​ （2）容斥原理：

​ Pr（A U B）= Pr（A）+ Pr（B）- Pr（A ∩ B）

​ 推广到三个时：Pr（A1 U A2 U A3）= Pr (A1) + Pr(A2) + Pr(A3) - Pr(A1∩ A2 ) - Pr(A1∩ A3 ) - Pr(A2 ∩ A3 ) + Pr(A1 ∩ A2 ∩ A3）

​ 一般式可以写成

​ （3）如果A ⊆B，那么Pr（A）≤Pr（B）。 Pr（B）= Pr（A）+Pr（B∩ A^c ）

​ 注：即便A是B的真子集，也不一定有Pr（A）＜ Pr（B）。

2.5.1 全概率公式

​ 第一条规则可以根据概率函数的性质来推导。

​ A ∩ A^c = Ø 。 A ∪ A^c = Ω

​ 则有：1 = Pr （Ω）= Pr（A ∪ A^c）= Pr（A）+ Pr（A^c）

​ 它虽然简单，但却很有用处。因为很多时候计算某件事不发生的概率往往比计算它发生的概率容易得多。

2.5.2 容斥原理

​ 为了证明第二条规则，我们可以利用这样一个事实：如果存在有限多个两两互不相交的集合Ci，那么这些集合的并的概率就等于它们各自概率的和（第三条概率公理）。

​ 这样，我们就可以把A∪B写成 A∪B= （ A\（A∩B））∪ （B\（A∩B））∪（ A∩B ）

​ 改写后的格式既没有重复计算，又考虑了所有可能的情况。也就是我们通过集合论符号本身已经可以解决这一问题。但当集合的个数增多时，我们很有可能会在查看符号的过程中迷失。

​ 从另一个角度来看，我们可以试着对公式的两边分别进行解释，再来证明它们描述的是同一件事。

​

​ 等式左端是A∪B的概率。换句话说，它是x属于A或者x属于B或者x既属于A又属于B的概率。

​ 等式右端的Pr（A）和Pr（B）分别表示x属于A和x属于B的概率。将其加和后，应该理解为x属于A或B其中之一的概率再加上x为重复部分元素的概率。因此，再减去x属于A ∩ B的概率后，我们就得到了x属于A或B其中之一的概率。
