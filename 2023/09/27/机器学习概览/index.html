机器学习概览

2023-09-27
Machine Learning
1.9k words
  • Machine Learning

人类学习利用经验不断提高性能，机器善于处理数据不断提高性能

机器学习

定义：

  机器利用数据学习人类经验，来不断提高自身性能的过程。

  目前主要研究智能数据分析的理论和方法，并已经称为智能数据分析技术的源泉之一。

   大数据并不等于大价值，需要通过智能数据分析来攫取更有价值的数据内容，也就是机器学习。

机器学习是人工智能的核心研究领域之一

  今天的“人工智能热潮”正是由于机器学习、尤其是深度学习技术取得了巨大进展，基于大数据，大算力发挥出了巨大威力。

典型的机器学习过程

  • 监督学习 supervised learning
  • 无监督学习 unsupervised learning
  • 强化学习 reinforcement learning

数据集：训练集、测试集

模型：决策树、神经网络、支持向量机、Boosting、贝叶斯网、……

  在训练集中，使用学习算法训练模型，模型用决策树等方式来搭建，模型输出的结果大致可分为 用于分类的分类型 和 用于输出具体数据的回归型。分类中还可以具体分为二分类，多分类等。

学习的目标

  机器学习希望从历史数据中学会推知未来的数据，也就是说，机器学习技术的根本目标就是模型具有泛化能力，即应对未见样本的预测能力。

Machine Learning = Task + Data + Objective + Algorithm

  • 任务通常可以指学习一个从特征空间到类别空间的映射 f：X→Y
  • 以西瓜任务为例：
    • X：西瓜的特征，如颜色、根蒂的蜷缩程度、敲击的声音等
    • Y：西瓜是好还是坏
  • f 通常来自一个约定好的空间

Task：

  • 分类任务：Y包含若干离散的属性值

    二分类：{0,1},K分类：{0,1, ……，k}

    如：垃圾邮件分类、动物识别、诈骗电话检测、动作分类。

  • 回归任务：Y通常是实数值

    如：外卖送达的预计时间，自动驾驶时方向盘旋转的幅度，刹车幅度。

  • 聚类任务：把数据集中的样本划分为若干个子集

    如：商品分类，用户分类

  • 降维任务：降低特征维度，用更少的特征表示数据

Data：

  • 训练数据：训练机器学习模型的基础资源
    • 监督学习：Dtr = {（x1，y1），（x2，y2），……（xn，yn）}
    • 无监督学习：Dtr = {x1，x2，……，xn}
    • 半监督学习：Dtr = {（x1，y1），（x2，y2），……（xt，yt），xt+1，……，xn}
  • 测试数据：评估模型的性能，在训练过程中无法见到
    • Dte = {（x1，y1），（x2，y2），……，（xm，ym）}
  • 验证数据：用于训练过程中辅助评估模型的性能

监督学习：所有的训练样本均有对应的标注

无监督学习：所有训练样本均没有标注

半监督学习：部分数据有标注，部分数据无标注

强化学习：标记信息延迟的监督学习问题

Objective:

  • 评价指标 / 损失函数：反映了模型 f 的性能好坏，用于指导模型训练

  • 分类问题：

    • 正确率：模型预测正确的概率

  • 回归问题

    • 均方误差（Mean Squared Error）

Algorithm:

机器学习算法𝒜

  • 输入：训练数据集Dtr，评价指标M（f）/损失函数loss（f），f的函数空间ℱ
  • 输出：学得的模型 f

学习算法运行的过程称为模型的训练过程

即，在所有可能的 f 组成的空间中进行搜索的过程

机器学习常用算法：

  • 线性回归算法
  • 逻辑回归算法
  • 支持向量机算法
  • k-近邻算法
  • k-Means算法
  • 决策树算法
  • 随机森林算法
  • 朴素贝叶斯算法
  • 神经网络

深度学习：通常是指以深度神经网络为模型的一类机器学习方法

深度学习最重要的作用：表示学习

  传统的机器学习方法通常需要人工地设计或选择特征，这个过程叫做特征工程。特征工程重要而耗时且困难、需要对数据有很好的理解和分析，而且不同的任务可能需要不同的特征。

  深度学习方法则可以自动地从原始数据中学习特征，这个过程叫做特征学习或表示学习。深度学习方法可以利用多层的非线性变换来逐层地提取出更高层次的特征，而不需要人工地指定或选择特征。

深度学习方法的优点：

  可以减少人工干预，提高模型的泛化能力，处理高维、复杂、非结构化的数据。

深度学习方法的缺点：

  需要大量的数据和计算资源，难以解释模型的内部机制，容易受到对抗样本等攻击。

泛化风险 与 经验风险：

  泛化误差，指在未来样本上的误差。经验误差，指在训练集上的误差，故也称为“训练误差”。泛化误差越小越好，但经验误差并不是越小越好。当经验误差过小时，可能会出现过拟合的情况。一般而言，训练样本越少，模型越复杂，越容易过拟合。

机器学习并不是“一切皆可学”

  • 特征信息不充分，重要特征信息没有获得时
  • 样本信息不充分，仅有很少的数据样本时

No Free Lunch定理

定理内容：一个算法𝒜1若在某些问题上比另一个算法𝒜2好，则必然存在另一些问题能使得𝒜2比𝒜1好

它有一个重要前提是，假设所有“问题”出现的机会相同，或所有问题同等重要

这也就是说，脱离具体问题，空泛地谈论“什么学习算法更好”没有意义，我们通常只需要关注自己正在试图解决的问题，需要具体问题具体分析。

机器学习与其他领域的关系

  • 机器学习与数据挖掘
    • 机器学习和数据挖掘都涉及数据分析
    • 机器学习、数据库、统计学是数据挖掘的关键支撑技术
    • 机器学习更偏技术方法，数据挖掘更偏应用
  • 机器学习与数据科学
    • 机器学习是数据科学实现智能化的关键步骤——数据智能分析
    • 数据科学还包括很多内容，例如收集、存储、传输、管理大数据
    • 大数据研究目的是为了利用大数据，没有机器学习提供数据分析技术，大数据利用无从谈起
  • 机器学习和 计算机视觉/自然语言处理/模式识别
    • 机器学习是计算机视觉的核心技术/自然语言处理/模式识别
    • 计算机视觉/自然语言处理/模式识别 是机器学习的重要应用
  • 机器学习和统计学
    • 统计学是机器学习的重要理论基础
  • 机器学习和神经科学
    • 机器学习发展过程中，经常受到神经科学思想的启发
    • 神经科学发展比较缓慢，不够成熟，机器学习通常借鉴其思想而不借鉴其技术
