优化问题

优化问题是指，在给定的约束条件下，寻找一个函数的最值的问题。它的定义如下：

minimize f0(x)

subject to fi(x)≤bi， i=1，……，m

其中，向量x=（x1，……，xn）称为问题的优化变量，函数f0：R^n→R称为目标函数，函数fi：R^n→R，i=1，……，m被称为不等式约束函数，常数b1，……，bm称为约束上限或者约束边界。如果在所有满足约束的向量中向量x*对应的目标函数值最小，即对于任意满足约束f1（z）≤b1，……，fm（z）≤bm的向量z，有f0（z）≥f0（x*），那么称x*为问题的最优解。

  在当前的学习内容中，我们所研究的是凸优化问题。一般来说，凸优化问题比非凸优化问题更简单。

  什么叫做凸优化问题呢，它是指目标函数为凸函数，而其约束条件是一个凸集或者约束条件是由若干个凸函数组成的。我们接下来讨论凸函数与凸集等内容。它们属于是数学预备知识。

1.1 仿射集 Affine Set

  它的定义是：如果一个集合C是仿射集，那么任意从集合C中取两点x1，x2，它们所确定的直线依旧在集合C中。

  我们考虑最简单的例子：设平面上的两点为x1，x2，它们所确定的一条直线为y=Θx1+（1-θ）x2（Θ∈R）。这条直线上任意两点所确定的直线即为同一条直线，所以这条直线是一个仿射集。但如果我们加入限制条件，令θ∈[0,1]，则这个集合变为了线段。其上任意两点所确定的直线反过来包含了原线段，故不为仿射集。

  任意一条直线，任意一个二维平面，任意一个线性方程组的解集都为仿射集。

线性方程组的解集是仿射集，并且任意仿射集也都能写成线性方程组的解集

Proof：第一个命题是容易证明的，下面证明第二个命题

    对于任意一个仿射集C，可以找到一个与之相关的子空间V，使得C = x0 + V，其中x0是C中的任意一个元素。然后，可以构造一个矩阵A，使得V是A的零空间。即对于任意的v∈V，有Av=0。这样，对于C中的任意元素x，都有Ax=A（v+x0）=Av+Ax0=Ax0，记作b=Ax0，则C={x|Ax=b}，也就是C是线性方程组Ax=b的解集。

1.2 仿射组合 Affine Combination

  仿射组合是一种特殊的线性组合，它要求系数之和为1。也就是说，如果有一组向量x1，x2，……，xn，那么它们的仿射组合是θ1x1+θ2x2+……+θnxn，其中θ1+θ2+……+θn=1。

1.3 仿射包 Affine Hull

一个集合C⊆R^n中所有点的仿射组合的集合叫做C的仿射包，记作affC：

affC={θ1x1 + θ2x2 + …… + θnxn|x1，x2……xn ∈ C，θ1 + θ2 + …… + θn=1}

对于任意集合A，增加最少的元素能够使得集合A变为仿射集B，则仿射集B是A的仿射包。也就是B是包含A的最小的仿射集。

1.4 仿射集的子空间

若C是一个仿射集，集合V = C-x0 = {x-x0|x∈C，∀x0∈C}是与C相关的子空间

这相当于将仿射集C基于元素x0做了一个平移，使之必过原点，并且得到的子空间是仿射集。

2.1 凸集 Convex Set

一个集合C是凸的，若C中任意两点之间的线段都在C中，即对∀x1，x2∈C，∀θ∈[0,1]都有θx1+（1-θ）x2∈C

仿射集都是凸集，凸集是仿射集的特例。

2.2 凸组合 Convex Combination

若 θ1 + θ2…… + θn = 1，θi ≥ 0，将θ1x1+θ2x2……+θnxn的点称为点x1，x2，……，xn的凸组合

凸组合与仿射组合的区别：在凸组合中，参数θ必须大于等于0

2.3 凸包 Convex Hull

一个集合C的凸包，是C中所有点的所有凸组合的集合，记作convC：

convC = {θ1x1 + θ2x2 + …… + θnxn|xi∈C，θi≥0，θ1+θ2+……+θn = 1}

一个集合的凸包，也就是包含这个集合的最小的凸集。凸集的凸包是它本身。

3.1 锥 Convex Cone

如果对∀x∈C，θ ≥ 0都有θx∈C，则称集合C是锥

锥必过原点，如在二维平面中一条以原点作为端点的射线是锥，由多条这样的射线构成的集合也是锥

3.2 凸锥 Convex Cone

对∀x1，x2∈C，θ1，θ2≥0都有θ1x1+θ2x2∈C，这样的集合C称为凸锥

也就是说，如果集合C既是凸集也是锥，那么这样的集合就叫凸锥

3.3 锥组合

若θ1，θ2，……θn≥0，θ1x1+θ2x2+……+θnxn的点叫做点x1，x2，……，xn的锥组合，也称为非负线性组合。

如果点xi在凸锥C中，那么xi的每个锥组合也都在C中

3.4 锥包

一个集合C的锥包是C中所有点的所有锥组合的集合，即：

{ θ1x1+θ2x2+……+θnxn|xi∈C，θi≥0 }

对于任意集合A，增加最少的元素使得A变为凸锥B，则凸锥B是A的锥包，即B是包含A的最小凸锥

4.1 凸函数的定义

  一个函数f：R^n → R是凸的，如果其定义域dom f是凸集，并且对于所有点x，y∈dom f，θ∈[0,1]，都有：f（θx+（1-θ）y）≤θf（x）+（1-θ）f（y）

  从几何意义上讲，凸函数图像上的任意两点连成的线段都位于图像的上方。

4.2 下/上水平集

函数f：R^n→R的α下水平集定义为：

Cα={x∈dom f|f（x）≤a}

实质上就是一个函数满足一定条件时的定义域上的一系列点的集合。

凸函数的任意α下水平集都是凸集。而即便某个函数的下水平集都是凸集，这个函数也不一定是凸函数。（如：f（x）=-e^x）

凸函数的α上水平集（即将≤改为≥）也是凸集。

5.1 范数的定义

  满足以下条件的函数f：R^n→R，domf = R^n称为范数：

  • f是非负的：对所有x∈R^n，都有f（x）≥0；

  • f是正定的：仅对x=0成立f（x）=0；

  • f是齐次的：对所有x∈R^n和t∈R，都有f（tx）=|t|f（x）；

  • f满足三角不等式：对所有x，y∈R^n，都有f（x+y）≤f（x）+f（y）；

范数用符号f（x）=||x||表示，范数是对向量x的长度的度量，我们可以用两个向量x，y的差异的长度来衡量这两个向量之间的距离，即||x-y||，因此||x||也可以理解为向量x到原点的距离。
