普林斯顿概率论读本学习-Chapter 4 条件概率、独立性和贝叶斯定理

2023-09-05
Math
5.7k words
  • 概率论

​ 电子版课本

  事件不是孤立的，某个事件的发生会对另一个事件发生的概率产生影响。在计算时，我们经常得到额外的信息；在已知其他事件发生的前提下，我们想知道某个事件发生的概率有多大。这就是所谓的条件概率。我们希望利用更多的基本结果来找到条件概率的简单公式，并为此花费了大量时间。这种花费是有充分理由的。

  在前期我们大部分的讨论围绕着如何合理地猜测答案，以及如何检验答案。在此之后才会谈到证明。

引子

  三名学生进入了一个房间，他们每个人头上都戴着一顶白色或黑色的帽子。每顶帽子的颜色都是通过抛掷一枚均匀的硬币来决定的，并且抛掷一枚硬币的结果对另一枚硬币没有任何影响。也就是说，对每个学生来说，他戴两种颜色帽子的概率是一致的。

  而这三名学生是相互独立的。每个人都能看到其他两人的帽子，但看不到自己的。除了游戏开始前的一个初始策略会话以外，任何形式的交流都是不被允许的。一旦有机会看到其他人的帽子，他们就必须马上猜出自己帽子的颜色或者弃权。如果至少有一个人猜对并且没有人猜错的话，他们就可以共享300万美元奖金。如果只有一个人说话且坚持说是白色，那么他们获胜的概率是50%。但事实上，他们还可以做得更好，也就是将自己获胜的概率提升到50%以上。具体的方法暂且按下不表，我们可以先自由思考一会儿，在4.2.2节中我们会再次讨论。

4.1 条件概率

  无论概率论课上还是现实中，我们都会想知道某个事件发生的概率。譬如学校某一天下雨的概率，一种计算方法是用学校所在地一年的下雨总数除以365，但这显然不是很好的方法，下雨的稠密程度和月份等常常有关系。我们可以将问题更确切化，譬如想知道9月份下雨的可能性。在已知某个事件的前提下，我们考察的焦点会集中在样本空间的一个子集上；这类概率称为条件概率，记作Pr（A|B），读作“已知B时A的概率”，也就是说它代表了在事件B已经发生的前提下，事件A发生的概率。

  条件概率随处可见，考虑下列可能的应用：

  **1>.**体育运动方面：在一场英式足球赛中，第75分钟时埃弗顿队以1-0落后，那他们获胜的概率有多大？

  **2>.**医疗保健方面：对于一个65岁且不吸烟的病人来说，他患心脏病的概率是多少？同岁数吸烟的人呢？已知前两种药物都无效，第三种药物有多大概率能使病人痊愈呢？一位病人的一种病症检测结果为阳性，他的确得病的可能性有多大？

  **3>.**政治方面：在民意测试中，一位候选人得到了53%的选票，那么她在最终选举中胜出的概率是多少？

  上述这些问题的共同之处在于：我们可以利用已知信息把一般的概率问题（她在最终选举中获胜的概率是多少？）转化成一个更具体的问题（已知民意测试的数据后，她在选举中获胜的概率是多少？）。在很多方面，条件概率都是最常见且应用最广泛的概率形式，因为我们总能得到一些可用的额外信息。

  有这样一个大转盘游戏：三个人依次转动大转盘，在这个转盘上有21个数，分别表示从0美元开始，以5美分为增量直到1美元的21个不同价格。每个玩家最多可以转动两次；且转动两次时得分为两次的结果之和。得分最接近且不超过1美元的玩家获胜。

  当我们作为第一个人上前转动时，第一次转动和第二次转动得到的结果，其概率分布是不一样的。如果第一次结果比较小，比如10，那么我们会尽量在第二次转动时得到90附近的数，以便能获得更大的得胜概率。而如果第一次的数比较大，那么我们有可能选择不再转动。如果我们是第三个人，我们的转盘情况也会受到前两位玩家的得分状况的影响：如果前两个玩家的得分都超过了1美元，那么第三个玩家就不需要转动两次，不可能超过1美元；但如果前两位玩家中出现了95分，那么第三个玩家就不得不转动两次，得分超过1美元的概率也会很大程度上提升。

4.1.1 猜测条件概率公式

  经过上面的讨论，我们现在已经明确了两件事： 1.条件概率是有用的，很多地方都会有所涉及 2.我们需要寻找一个计算条件概率的方法。

  事实上，不仅是方法，经历了桥牌中的复杂计数，我们更想找一个更加系统的好方法。

   我们可以尝试找到这个公式的一些特征，并预测它的可能形式。

  Pr（A|B）当然与Pr（A）和Pr（B）有关。我们可能会想找到一个合适的F，使得Pr（A|B）=F（Pr（A），Pr（B））。但我们试着举出一些极端例子就会发现，这是没有办法做到的。极端情况也就是说A与B充分接近或者A与B充分远离，并且将A和B的概率赋为好计算的值。

  我们可以给出一个情形：Pr（A）=Pr（B）=Pr（B^c）。也就是说三个事件的概率均为1/2.

  如果存在这样的F，就将有：1 = Pr（B|B）=F（1/2,1/2）=Pr（B^c,B）=0。

  也就是说，Pr（A|B）不可以简单地取决于Pr（A），Pr（B）的数值，而应该与A和B的本质有关。或者保守些讲，至少与Pr（A）和Pr（B）以外的东西有关。

  我们再想一想Pr（A|B）的定义是已知B发生的情况下A发生的概率，那至少应该有Pr（A∩B）项。还有一些特殊的取值：Pr（B|B）=1；Pr（B^c,B）=0；0≤Pr（A|B）≤1。这时我们可以猜测一个简单的形式：Pr（A∩B）/Pr（B）（需要注意，分母并不一定非0，当Pr（B）=0时，Pr（A∩B）也为0，这会让算式成为0/0型不定式）。除此之外，A=B时，Pr（A|B）=1，A+B^c时，Pr（A|B）=0。同时，由于A∩B⊆B，也能保证概率始终介于0和1之间。

4.1.3 Veen图法

  我们还可以再举一些条件概率的例子。

​ 例子：在某个城镇里，一个成年人拥有汽车或房子的概率是90%。如果有70%的成年人拥有汽车，有50%的成年人拥有房子，那么一个拥有房子的成年人也拥有汽车的概率是多少？

​ 解答：首先，我们需要考虑我们真正想要找到的是哪个概率。用事件A表示拥有一辆汽车，用事件B表示拥有一套房子，那么我们想求的概率就是Pr（A|B）。我们已知Pr（B），再求出Pr（A∩B）即可。

  Pr（A∪B）=Pr（A）+Pr（B）-Pr（A∩B）。所以Pr（A∩B）=0.3。则Pr（A|B）=Pr（A∩B）/Pr（B）

=0.3/0.5=0.6。

4.2 一般乘法法则

4.2.1 陈述

我们很容易得到另一个结论，它被称为一般乘法法则：

一般乘法法则：

​ Pr（A∩B）=Pr（A|B）*Pr（B）

  不过，虽然我们通过条件概率公式得出这个公式，但实际上这个公式的适用范围更大：当B的概率为0时依然成立。

  这样以来，当条件概率和B的概率都很容易计算时，利用一般乘法法则就很容易能求得A和B同时发生的概率。

4.2.2 帽子问题与纠错码

  开篇我们提到的帽子问题的解决策略为：当且仅当每个人在看到两顶相同颜色的帽子时才开口说话，并且要说相反的颜色。

  为什么要这样呢？

  帽子的颜色共有8种可能的分法：WWW、WWB、WBW、BWW、WBB、BWB、BBW和BBB。三顶帽子的颜色完全相同的情况仅有两种。在6种有两顶相同帽子的情况里，只有一个人能看到两顶相同颜色的帽子，这个人说出与自己看到的颜色相反的颜色，那么我们就能够猜对8种情况里的6种，这样可以使得胜率为75%.

4.2.3 高等注解：条件概率的定义

  我们只是根据逻辑和推测给出条件概率的公式为Pr（A|B）=Pr（A∩B）/Pr（B）。它是否可以通过前面的概率公理推导出来呢？是可以的。我们可以尝试做成这件事。

  假设我们考察的概率空间是（Ω ，∑，P），但事件B的发生使得概率分布变成了PB，那么现在事件A发生的概率应该是多少呢？我们可以得到两个合理的假设：首先，因为B已经发生，所以PB（A）=0对所有的A⊆ B^c均成立；其次，我们不希望概率的相对大小发生变化。也就是说，如果A，C⊆Ω与B有相交，那么PB（A）/PB（C）=P（A∩B）/P（B∩C）。不妨取C=B，则有PB（C）=1，P（B∩C）=P（B）。也就是PB（A）=P（A∩B）/P（B∩C）。

4.3 独立性

  通俗地讲，如果两个事件不相互影响，那么就说它们是相互独立的。换句话说，从某个事件发生无法推出另一个事件发生的可能性。

  （暂时定义）对于事件A和B，如果Pr（A|B）=Pr（A），那么我们说A和B相互独立的。也就是说，如果事件B的发生不会改变事件A发生的概率，那么A和B就是相互独立的

  这个定义紧扣了独立性的核心内容，也就是Pr（A|B）=Pr（A），这说明由事件B的发生不能得到有关A是否发生的任何信息。

  此时，我们再使用一般乘法法则，可知对于相互独立的事件A和B，有Pr（A∩B）=Pr（A）*Pr（B）

  但我们说这个定义是“暂时的”。因为存在Pr（B）为0时的情况，不适用于上面的定义。以及没有体现出A和B的对称性；毕竟A与B是相互独立事件时，B与A显然也是。

​ 独立性（两个事件）：如果事件A和B满足，Pr（A∩B）=Pr（A）*Pr（B），那么A和B就是独立的

​ 独立性（三个事件）：如果事件A，B和C满足

  1. Pr（A∩B∩C）=Pr（A）*Pr（B）*Pr（C）

  2. 其中任意两个事件都是独立的

    那么A、B和C就是相互独立的

​ 独立性（一般情形）：如果事件A1，……，An满足

​ 1.Pr（A1∩……∩An）=Pr（A1）……Pr（An）

​ 2.{A1，……，An}的任意一个非空子集都是相互独立的，那么A1，……，An就是相互独立的

​ 当然，即便任意两个事件都独立，三个或更多事件也可能会相互依赖。

  例如： 记事件A表示第一次抛掷出了一个偶数；事件B表示第二次抛掷出了一个偶数，事件C表示两次掷骰子的点数之和是个偶数。

​ Pr（A）=1/2，Pr（B）=1/2，Pr（C）=1/2；

​ Pr（A∩B）=Pr（A）*Pr（B）=1/4

​ Pr（A∩C）=Pr（A）*Pr（C）=1/4

​ Pr（B∩C）=Pr（B）*Pr（C）=1/4；

  但是，Pr（A∩B∩C）≠ Pr（A）*Pr（B）*Pr（C），这是因为当A、B同时发生时，C必然发生，也就是Pr（A∩B∩C）= Pr（A）*Pr（B）

  在这里，尽管三个事件中的任意两个是相互独立的，但它们三个事件间存在依赖关系，即当A和B发生时，C就必然会发生

4.4 贝叶斯定理

  我们已经知道了一般乘法法则，即 Pr（A∩B）=Pr（A|B）*Pr（B），由于这里的A和B是等价的，我们还可以知道：Pr（A∩B）=Pr（B|A）*Pr（A）。而A ∩ B和B ∩ A又是同一件事，我们就可以得到：

贝叶斯定理：

  由一般乘法法则可以推出：对于事件 A 和 B，有

      Pr（B|A）*Pr（A）=Pr（A|B）*Pr（B）

  也就是说，只要有Pr（B）≠0，我们就会有：

      Pr（A|B）=Pr（B|A）*Pr（A）/Pr（B）

我们可以来看一些例子：

  Example 1：

   题面：假设有40%的成年人养狗，而每年有20%的养狗人会得流感。如果每年有15%的成年人会得流感，那么已经得流感的某个人养狗的概率是多少？

   解：用D表示某人养狗，P（D）=0.4；用F表示某人得了流感，P（F）=0.15；且P（F|D）=0.2；代入公式知Pr（D|F）=Pr（F|D）*Pr（D）/Pr（F）=0.2*0.4 / 0.15= 8/15.

  Example 2：

   题面：在全国范围内，每 15000 个人中就有一个人感染结核病。假设你所在的城镇出现了一次结核病恐慌；为了简单起见，我们假设该城镇的结核病发病率与全国平均水平相同。为了安全起见，你去医院检查是否感染了这种疾病。医生告诉你，这个测试出现假阳性的概率为1%，也就是说，每100个健康人中会有一个人的测试结果显示为阳性。医生还说，这个测试的假阴性率为0.1%，也就是说，每1000个得病的人中只有一个人的检测会显示为阴性。假设你检测的结果为阳性，那么得病的概率是多少？

   解：设D表示得结核病，P（D）=1/15000，设P表示检测结果为阳性。则P（P|D）=999/1000.我们还需要求出P（P）.事实上，只有得病和健康两种情况，并且这两个事件的交集为空。阳性的检测结果分别来自他们。即：P（P）=（1-P（D））*（1/100）+P（D）*（999/1000）。

     由此，我们可以代入贝叶斯定理：P（D|P）=P（P|D）*P（D）/P（P）≈0.0066.

     这个概率大概比大部分人的料想都会小很多。事实上，由于得病的几率较小，虽然健康人中出现假阳性的概率很小，但是与发生的概率相乘后，大小也会与得病的几率相差不多甚至超过得病的几率。

划分问题：

  划分成两部分：设B是满足0＜P（B）＜1的事件（即B^c也满足该条件）。因此，我们可将事件A划分成两个互不相交的事件，即A∩B和A∩B^c，而且有

  Pr（A）=Pr（A|B）P（B）+Pr（A|B^c）Pr（B^c）

  它清晰而容易理解，只需要知道A的总概率包括B发生时A的概率和B不发生时A的概率。

  Example：有两个装着红球和蓝球的罐子。在第一个罐子中，红球的比例是40%；在第二个罐子中，红球的比例是20%。你的朋友随机地从一个罐子中取出了一个红球。那么，这个红球取自第一个罐子的概率为多少？

  解答：我们要计算的是Pr（从第一个罐子中|红球）=Pr（第一个罐子）*Pr（红球|第一个罐子）/Pr（红球）

    随机从罐子里取球，所以从第一个罐子中取球的概率为0.5。另外，已知从第一个罐子取球时，取得红球的概率为0.4。

    Pr（红球）=Pr（A）*Pr（红球|A）+Pr（B）*Pr（红球|B）

         =0.5*0.4+0.5*0.2 = 0.3

所以可得Pr（从第一个罐子中|红球）=（0.5*0.4）/0.3=0.67。

4.5 划分和全概率法则

划分：

样本空间S的一个划分就是满足下列条件的可数个集合 { A1，A2，……}

  （1）如果 i ≠ j，那么Ai和Aj不相交。我们通常用 Ai ∩ Aj = ∅来表示这两个集合的交是空集。

  （2）全体Ai的并就是整个样本空间： UiAi=S

   从本质上看，划分就是一簇集合。在这簇集合中，每种可能的结果都恰好出现一次。在之前我们已经见过将样本空间划分为B和B^c，这里是更加一般地情况。

   关于符号，我们可以使用∑n来表示对于给定的集合{B1,B2,……}，将其所有的项加起来。而非常正式的写法是：用{Bn}n∈I来表示划分的元素，再用∑n∈I Pr（A|Bn）*Pr（Bn）来表示和。通常，我们使用前者。

全概率法则

全概率法则：如果{B1，B2，……}构成了样本空间S的一个划分（分成了至多可数个部分），那么对于任意的A⊆S，我们有

  Pr（A）=∑ n Pr（A|Bn）*Pr（Bn）

 对于所有的n，都应该有0＜Pr（Bn）＜1，否则条件概率就是无定义的。左侧不可取等是因为当Pr（Bn）=0时，Pr（A|Bn）无意义；右侧不可取等是因为当Pr（Bn）=1时，Pr（A|Bn）*Pr（Bn）=Pr（A），使得其它所有项为0。

Example：

  根据一些研究，在美国大约有30%的孕妇采用剖腹产，此时有0.2%的新生儿无法存活。在不采用剖腹产时，婴儿的存活率是99.1%。求新生儿的存活率是多少？

  解：用A来表示新生儿能够存活，B1表示采用剖腹产，B2表示采用非剖腹产。

  则有：Pr（A）=∑（i=1，n）Pr（A|Bi）*Pr（Bi）

        =Pr（A|B1）*Pr（B1）+Pr（A|B2）*Pr（B2）

        =（1-0.002）*30+（0.991）*（1-0.30）

        = 0.9931 = 99.31%

4.6 回顾贝叶斯定理

我们有了全概率公式，就可以给出贝叶斯定理的另一个形式：

贝叶斯定理：

设{Ai}（i=1，n）是样本空间的一个划分，那么

  Pr（A|B）=Pr（B|A）*Pr（A）/∑（i=1，n）Pr（B|Ai）*Pr（Ai）

Example：

  近年来，下雨的比赛日平均占到了30%。在下雨的日子里，85%的天气预报预测会下雨。在不下雨的日子里，只有20%的天气预报预测会下雨。某个比赛日，天气预报已经会下雨，那么实际下雨的概率是多少？

  解：用A1表示比赛日会下雨事件，A2表示比赛日不下雨的事件，B表示天气预报预测会下雨的事件。

    欲求Pr（A1|B）

    Pr（A1|B）=Pr（B|A1）*Pr（A1）/∑（i=1，n）Pr（B|Ai）*Pr（Ai）

          =Pr（B|A1）*Pr（A1）/（Pr（B|A1）*Pr（A1）+Pr（B|A2）*Pr（A2））

          =0.85*0.30/（0.85*0.30+0.20*（1-0.30））

          ≈0.645 569
