矩阵与方程组

  数学中最重要的问题之一是求解线性方程组。超过75%的科学研究和工程应用中的数学问题，在某个阶段都涉及求解线性方程组。我们有许多新的数学方法，可以将较为复杂的问题化为线性方程组。

1.1 线性方程组

定义

形如

​ a1x1 + a2x2 + …… + anxn = b

的方程称为含有 n 个未知量的线性方程，其中 a1 ， a2 ， …… ， an和b为实数，x1，x2，……，xn称为变量。

  含有 m 个方程和 n 个未知量的线性方程组（又称为 “m*n的线性方程组”）定义为

​ a11x1 + a12x2 + …… + a1nxn = b1

​ a21x1 + a22x2 + …… + a2nxn = b2

​ ……

​ am1x1 + am2x2 + …… + amnxn = bm

其中 aij 以及 bij 均为实数。

线性方程组的解

  • 定义：若有序n元组（x1，x2，……，xn）满足方程组中的所有方程，则称其为m*n的方程组的解

  • 存在性：有的方程组有解，有的方程组无解。无解的方程组称为是不相容的，同时，只要线性方程组存在解，那么就称它是相容的。

  • 解集：线性方程组的所有解的几何称为方程组的解集。相容方程组的解集非空，不相容方程组的解集为空。

    m*n的方程组可能相容，也可能不相容。对于相容的方程组来说，它们解集的情况只可能是 有且仅有一个一个 或者 无穷个 解。关于其原理，我们在之后再进行证明。现在先主要着眼于求所给方程组的所有解。

  • 等价方程组

      有这样两个方程组

    （a）3x1 + 2x2 - x3 = -2 （b）3x1 + 2x2 - x3 = -2

    ​ x2 = 3 -3x1 - x2 + x3 = 5

    ​ 2x3 = 4 3x1 + 2x2 + x3 = 2

      我们不难发现，方程组（a）更容易求解。而（b）实际上与（a）有相同的解，这通过三个方程之间进行运算容易得到。但（b）求解过程相对更复杂。

    定义：若两个含有相同变量的方程组具有相同的解集，则称它们是等价的

    下面的三种运算可能得到等价的方程组

      Ⅰ.交换任意两个方程的顺序

      Ⅱ.任一方程两边同乘一个非零的实数

      Ⅲ.任一方程的倍数加到另一方程上

    因此，对于给定的方程组，我们可以反复利用上面三个运算，来得到一个更加容易求解的等价方程组。

  • n * n 方程组

      接下来我们来讨论 n*n方程组。我们将证明，若n*n的方程组仅有一个解，则我们可以利用上面的运算Ⅰ和Ⅲ来得到一个等价的“严格三角形方程组”。

    严格三角形方程组的定义：若方程组中，第k个方程的前（k-1）个变量的系数均为零，且xk（k

    =1，……，n）的系数不为零，则称该方程组为严格三角形的

    n*n方程组的求解：只需由靠后的方程依次与前面的方程组做和或差消去未知数即可，较为方便。具体来讲，由第n个方程解得xn，将其代入第（n-1）个方程解得Xn-1，再将xn和xn-1代入第（n-2）个方程，解得xn-2，以此类推，称这种求解严格三角形方程组的方法为回代法

      一般地，给定一个n个方程n个未知量的线性方程组，可用运算Ⅰ和Ⅲ尽可能将其转化为等价的严格三角形方程组。事实上，当方程组不是唯一解时，不可能将其化简为严格三角形式。

    方程组的系数矩阵

       简单地讲，矩阵是一个矩形的数字阵列。一个m行n列的矩阵称为m×n矩阵，如果矩阵的行数等于列数，即m=n，则称该矩阵为方阵。

      将方程组的系数作为元，写成m×n的矩阵，这个矩阵就是方程组的系数矩阵。

       系数矩阵并不足以完全描述方程组，如果在系数矩阵的右侧添加一列方程组的右端项，又可以得到一个新的矩阵。这个矩阵就称为是方程组的增广矩阵。

    而如果是将一个m×r的矩阵B添加到一个m×n的矩阵A的右端时，相应的增广矩阵记为（A|B）。

​   每一个方程组都对应一个增广矩阵。在求解时，我们可以使用对增广矩阵进行运算的方法。之前我们提到的三个用于寻找等价方程组的运算，可以对应于下列增广矩阵的行运算。

初等行运算

   Ⅰ.交换任意两行

​    Ⅱ.以非零实数乘以某行

​    Ⅲ.将某行乘以某个实数再加到其它行

   我们的目标是得到一个严格三角形的矩阵，只需要依次使用每一列中某个选定的数字将同列的行数比它大的其他数字全部消去即可。

   在上述过程中，如果出现某一列的所有元素皆为0时，就需要考虑将其化为某种特殊的梯形或者是阶梯形。

1.2 行阶梯形

  我们已经提到过，在将矩阵化为严格三角形的过程中，需要多次使用行初等变换。但如果在某一步中可供选择的元素全为0，这个方法就无法进行了。此时的办法只能是跳过当前列并继续后续的过程，而处理完所有列以后所得到的矩阵我们称为阶梯形增广矩阵。

定义：

若一个矩阵满足

  1. 每一非零行中的第一个非零元为1；
  2. 第k行的元素不全为0时，第 k+1 行首变量之前零的个数多于第k行首变量之前零的个数；
  3. 所有元素均为0的行必定在不全为0的行之后，

则将其称为行阶梯形矩阵。

利用行运算Ⅰ、Ⅱ和Ⅲ，将线性方程组的增广矩阵化为行阶梯形的过程称为高斯消元法。

方程组解的情况：

  1. 若方程组相容并且行阶梯形矩阵的非零行构成了严格的三角形，则这个方程组有唯一解。
  2. 若一个线性方程组中方程的个数多于未知量的个数，则称其为超定的，超定方程组通常是（但不总是）不相容的。
  3. 若化简后的方程组中出现系数全为零而常数项不为0的行，则说明其为不相容的。

超定方程组、亚定方程组：

  1. 超定方程组：若一个线性方程组中方程的个数多于未知量的个数，则称其为超定的，超定方程组通常是（但不总是）不相容的。
  2. 亚定方程组：若一个线性方程组中未知量的个数多于方程的个数，则称其为亚定的，亚丁方程组有可能不相容，但大部分情况下都是相容的且解的个数无穷。它不可能有唯一解；用上面的结论来说，这是因为它行数和列数的限制，系数矩阵不可能构成严格的三角形。

行最简形：

定义：若一个矩阵满足：

  1. 矩阵是行阶梯形的
  2. 每一行的第一个非零元是该列唯一的非零元

  则称该矩阵为行最简形

齐次方程组：

如果线性方程组的右端项全部为零，则称其为齐次的。齐次方程组总是相容的，并且容易发现有一个解为零向量。因此，如果一个线性方程组有唯一解，那这个解必然是平凡解（0,0,0，……，0）

定理：若n>m，则m×n的齐次线性方程组有非平凡解。

  齐次方程组总是相容的。因为其增广矩阵的行阶梯形最多有m个非零行，也就是最多有m个首变量。因为总变量个数n满足n＞m，则必然存在自由变量。而自由变量可以任意取值，且自由变量的任意一组取值都可以得到方程组的一组解。

1.4 矩阵代数

 实数的代数法可能使用于矩阵，也可能不适用，例如矩阵加法满足交换律，但矩阵乘法却并不。

代数法则

定理 1.4.1

在定义了需要的运算以后，下面的法则对于任意的标量α和β以及矩阵A，B和C都是成立的

  1. A + B = B + A
  2. （A+B）+ C = A +（B+C）
  3. （AB）C=A（BC）
  4. A（B+C）=AB + AC
  5. （A+B）C=AC+BC
  6. （αβ）A=α（βA）
  7. α（AB）=（αA）B=A（αB）
  8. （α+β）A=αA+βA
  9. α（A+B）=αA+αB

单位矩阵

  矩阵乘法中存在单位元，我们把这个矩阵记作I，满足 IA=AI=A。

定义：单位矩阵

  n*n的单位矩阵为矩阵I=（δij），其中 δ ij = 1（当i == j时）0（当i != j时）

  n*n单位矩阵I的列向量是用来定义n维欧几里得坐标空间的标准向量。I的第j列向量的标准记号为ej，而不是通常的ij。也就是说，n*n单位矩阵可写为 I=（e1，e2，……，en）

矩阵的逆

  实数乘法中，一个数字的倒数（如果存在）为它在乘法中的逆元。而矩阵乘法中有单位元，对于一部分矩阵，也存在乘法的逆元。

定义：矩阵的逆

  若存在一个矩阵B使得AB=BA=I，则称n*n的矩阵A为非奇异的或者可逆的。矩阵B称为矩阵A的乘法逆元。

  一个矩阵最多有一个乘法逆元，这是因为若B和C均为A的乘法逆元，则有 B = BI = B（AC）=（BA）C = IC = C。

  我们将非奇异矩阵A的乘法逆元简称为A的逆，并记为**A^(-1)**。

定义：奇异矩阵

  一个n*n矩阵若不存在乘法逆元，则称为奇异的

Notice：只有方阵才有乘法逆元。对于非方阵，不可以说它是奇异或者非奇异的。

  我们通常使用非奇异矩阵的乘积。并且，任意非奇异矩阵的乘积也是非奇异的。

定理

  若A和B为非奇异的 n*n矩阵，则AB也必然为非奇异的，并且（AB）^(-1)=B^(-1)A^(-1)

Proof： （B^(-1)A^(-1)）AB=B^(-1)(A^(-1)A)B=B^(-1)B=I

转置的代数法则
  1. （A^T）^T = A
  2. （αA）^T = αA^T
  3. （A+B）^T = A^T + B^T
  4. （AB）^T = B^TA^T

前三个是显然的，我们尝试证明第四个法则成立。

Proof：只需证明（AB）^T=B^T*A^T即可

   （AB）^T的（i，j）元素为AB的（j，i）元素，该元素为A的第j个行向量与B的第i个列向量相乘求得，也就是（j，i）= aj1b1i + aj2b2i + …… + ajnbni

   B^TA^T的第（i，j）元素为B^T的第i个行向量与A^T的第j个列向量的乘积，也就是B的第i个列向量和A的第j个行向量的乘积，也就是（i，j）= b1iaj1 + b2iaj2 + …… + bniajn

   因此，（AB）^T与B^TA^T相等

  这也就是说，如果矩阵B可以由矩阵A经过有限次行运算得到，则B与A是行等价的。特别的，当且仅当Ax=b 和 Bx=c的等价方程组时，两个增广矩阵（A|b）和（B|c）是行等价的

性质：行等价矩阵

  Ⅰ.对称性：若A与B是行等价的，则B与A是行等价的

  Ⅱ.传递性：若A与B是行等价的，且B与C是行等价的，则A与C是行等价的

非奇异矩阵的等价条件

  （a）A是非奇异的

  （b）Ax = 0仅有平凡解0

  （c）A与I行等价

Proof：

（1）a→b

   若A是非奇异的，且列向量x是Ax=0的一个解，则x=Ix=（A^(-1)A）x=A^(-1)（Ax）=A^(-1)0=0

（2）b→c

   我们假设b中方程组在有限次行初等变换后变为 Ux=0，且U为行阶梯形矩阵。假设U最下方有至少一行元素全为0，那么这个方程组就满足未知数数量大于方程数量。因此必然有非零解。这与b本身出现了矛盾。因此，这个U必然为一个对角元素全为1的严格三角形矩阵，也就是说明了A与I等价。

（3）c→a

   这是显然的，因为A=EkE（k-1）……E1I。

当且仅当 A非奇异时，n个未知量n个方程的线性方程组Ax=b有唯一解

Proof：

  必要性：若A为非奇异的，并且x为Ax=b的一个解，即Ax=b，将其两端同乘A^(-1)，可知x必等于A^(-1)b

  充分性：若Ax=b有唯一解，设其为x，假设A为奇异的，则方程Ax=0应该有一个解z≠0，也就是Az=0，也就是说A（x+z）=Ax+Az=b+0=b，说明x与（x+z）同时为Ax=b的解，与唯一解矛盾，舍去。则A为非奇异矩阵。

对角矩阵和三角形矩阵

  一个n*n矩阵A，当i＞j时，aij = 0，则称为上三角形的；当i＜j时，aij = 0，则称为下三角型的。上三角形的和下三角型的统称为三角形的。

  三角形矩阵的对角元素可能为0。但对严格三角形的线性方程组Ax=b，系数矩阵A必为对角元素非零的上三角形矩阵。

  一个n*n的矩阵A，当 i ≠ j 时，aij=0，则称为对角的。显然，对角矩阵既是上三角的又是下三角的。

  除此之外，如果一个三角形矩阵的对角线均为1，我们可以称这样的矩阵为单位上/下三角形矩阵
